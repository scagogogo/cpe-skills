# Changelog

## v0.1.0 (2026-06-14)

### Added

- CPE 2.2 and 2.3 URI parsing and formatting
- WFN (Well-Formed Name) binding and unbinding
- NISTIR 7696 name matching engine
- NISTIR 7695 character escaping system
- Builder pattern for CPE construction
- CPE generation, fuzzy generation, and merging
- Component type system with validation and normalization
- Version comparison and range matching
- CPE applicability language (AND/OR expressions)
- In-memory and file-based storage backends
- NVD CPE dictionary integration
- CVE-CPE mapping and querying
- CPE set operations (union, intersection, difference)
- Advanced matching with regex, partial, subset, superset, and distance modes
- CLI tool for CPE operations
- Comprehensive convenience functions (MustParse, QuickMatch, Clone, etc.)
- 100% test coverage with 976+ test cases
- GitHub Actions CI/CD with goreleaser
